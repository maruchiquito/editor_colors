Editor de Imagen con Reducción de Colores (HTML + Canvas)

Este proyecto es un editor de imágenes en HTML, CSS y JavaScript puro que permite cargar una imagen, analizar sus colores y reducir la paleta de colores a un número definido por el usuario, todo directamente en el navegador sin librerías externas. La aplicación utiliza el elemento <canvas> para procesar píxeles, detectar colores únicos, calcular su frecuencia y aplicar una reducción real de colores basada en la distancia RGB más cercana.
